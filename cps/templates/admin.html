Disable email?

Turning off this setting will erase your email server configuration. You will need to re-enter your settings if you wish to re-enable this feature.

Google drive support is experimental.

Books stored in Google Drive will be placed in a user visible folder. All changes to your library must be made via Bookbag to avoid broken links and possible data loss. Do not move, rename, or delete files directly in Google Drive.

If you would like to enable Google Drive book storage anyway, please follow the instructions below to get started:

  1. Go to the Google Cloud Console and create a project
  2. Enable the Google Drive API for your project
  3. Go to Credentials and create an OAuth 2.0 Client ID (Web Application type)
  4. Add your Bookbag URL as an authorized redirect URI
  5. Download the credentials and save the file as client_secrets.json
  6. Place client_secrets.json in Bookbag's config directory
  7. Restart Bookbag and return to this page
